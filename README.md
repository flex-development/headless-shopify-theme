## Getting Started

### Installation

1. Download a copy of the theme repository

2. Upload `headless-shopify-theme.zip` to Shopify

3. Enable the theme

4. Create a page that uses the `page.api.menus` template

   - The page `handle` will be the API endpoint used to access the shop's menus

To extend store functionality, install [Accentuate Custom
Fields](https://www.accentuate.io/), or a similiar metafields plugin.
